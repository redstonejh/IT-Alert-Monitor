1 Threat base score ESET threat keywords set the starting score. Ransomware, backdoor, credential theft, trojan, phishing, downloader, riskware, and PUA each carry different base weight.
+
2 Subsequent event increases Repeated hits on the same host raise the score each time. Spread across endpoints, multiple days, faster velocity, and host alert volume can add more weight.
=
3 Resolution adjustment Failed, unresolved, not cleaned, or action required increases severity. Cleaned, deleted, quarantined, blocked, removed, or terminated discounts the score.
